# ClassTools

这是一个用在班级电脑上面的教室系统🎉，以动态壁纸方式呈现高考倒计时，作业，课程表，值日生等看板。

本项目使用 Electron Vue3 [NaiveUI](https://www.naiveui.com) 等工具构建。

[2.0版本](https://github.com/luoling8192/ClassTools)

## 实现功能
- [x] 时钟显示
- [x] 高考倒计时
- [x] 实时天气
- [x] 作业布置
- [x] 独立作业看板（晚自习可用）
- [x] 标语展示
- [x] 值日生设置
- [x] 课程表展示
